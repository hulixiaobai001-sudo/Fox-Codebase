## 🔫 西部对决 · Western Showdown

西部主题轮盘赌双人对战游戏。
A Wild West themed Russian roulette duel game.

**🎮 在线游玩 | Play Online**
👉 **https://hulixiaobai001-sudo.github.io/Fox-Codebase/**

### 三种模式 | Three Modes

| 模式 | 说明 |
|------|------|
| 🏠 本地 Local | 双人同屏 Hot-seat |
| 🤖 人机 AI | 三档难度 + 自定义参数 |
| 🌐 联机 Online | WebSocket 中继，任何网络都能连 |

### 游戏特色 | Features

| 中文 | English |
|------|---------|
| 6发弹巢轮盘赌 | 6-chamber revolver roulette |
| 10种道具系统 | 10 unique items |
| 暴击 & 决斗系统 | Critical hits & duels |
| 30秒倒计时 | 30-second turn timer |
| 150个成就 | 150 achievements |
| 5套主题色 | 5 themes |
| 自定义规则投票 | Custom rule voting |
| 观战模式 | Spectator mode |
| 聊天 + 表情包 | Chat & emoji picker |
| 键盘快捷键 | Keyboard shortcuts |
| DeepSeek AI 接入 | DeepSeek API for AI |

### 联机服务器 | Online Server

`server/index.js` — WebSocket relay server on Railway.

**功能 | Features:**
- 房间管理 Room management
- 消息中继 Message relay
- 全局聊天 Global chat
- 随机匹配 Matchmaking
- 观战中继 Spectator relay
- DeepSeek API 代理 Proxy (`/api/chat`)
